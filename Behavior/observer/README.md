옵서버 패턴(observer pattern)은 객체의 상태 변화를 관찰하는 관찰자들,  
즉 옵저버들의 목록을 객체에 등록하여 상태 변화가 있을 때마다 메서드 등을 통해,  
객체가 직접 목록의 각 옵저버에게 통지하도록 하는 디자인 패턴이다.
